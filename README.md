# ergodox
My Ergodox configurations

## Original
### Configurator:
https://keyboard-configurator.massdrop.com/ext/ergodox
### Firmware loader
* http://www.pjrc.com/teensy/loader_linux.html
* Run `teensy`
* Drag/drop .hex file
* Click "auto" button
* Press Teensy button on keyboard
